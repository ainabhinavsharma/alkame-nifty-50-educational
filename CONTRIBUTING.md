## \# Contributing to alkame-nifty-50-educational

## 

## \## How to Contribute

## 1\. Fork this repository to your personal GitHub account

## 2\. Clone your fork locally

## 3\. Create a feature branch: `git checkout -b feature/your-feature-name`

## 4\. Make your changes and commit with clear messages

## 5\. Push to your fork and open a Pull Request against `Alkameinc:main`

## 

## \## Branch Naming Convention

## \- `feature/` — new features or upgrades

## \- `fix/` — bug fixes

## \- `docs/` — documentation updates

## 

## \## PR Requirements

## \- Describe what you changed and why

## \- Reference the relevant task/issue number if applicable

## \- Ensure code runs without errors before submitting
